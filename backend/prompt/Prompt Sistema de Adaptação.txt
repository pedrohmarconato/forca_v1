# PROMPT DE ADAPTAÇÃO DO SISTEMA

Este é um prompt de fallback para o Sistema de Adaptação. Este prompt é usado quando o prompt original não pode ser carregado.

O objetivo deste prompt é adaptar os planos de treinamento baseado em:
- Humor do usuário (muito cansado a muito disposto)
- Tempo disponível (muito curto a muito longo)
- Restrições temporárias (lesões, etc)
