# PROMPT DO TREINADOR ESPECIALISTA

Este é um prompt de fallback para o Treinador Especialista. Este prompt é usado quando o prompt original não pode ser carregado.

O objetivo deste prompt é gerar planos de treinamento personalizados com base nos dados do usuário. Inclui:
- Análise do perfil físico
- Avaliação de objetivos
- Criação de planos adaptados a restrições
- Definição de microciclos e sessões de treino
